# 用户中心
一个后台管理系统，对用户进行统一的管理。

## 技术选型
### 前端
1. React
2. Ant Design Pro + Ant Design + Ant Design components

### 后端
1. Spring Boot
2. Mybatis + Mybatis Plus
3. MySQL
4. knife4j

## 前端优化点
1. 全局响应拦截器

## 后端优化点
1. 通用返回类
2. 自定义异常
3. 捕获全局异常
4. 跨域处理
5. 多环境配置

## 部署
1. 原生部署
2. 宝塔面板部署
3. docker 部署
